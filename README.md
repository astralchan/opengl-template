# About
A basic reference for when I forget how to set up a rendering context in C with
GLFW and GLEW.
# Building
## Dependencies
* glew-devel
* glfw-devel
* openGL drivers - mesa, nouveau, etc
* pkg-config
* C compiler (e.g. gcc)
* make
## Build
To build, just run
```
make
```
Then, to run the program:
```
./prog
```
